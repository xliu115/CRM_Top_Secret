# Project Agents

This repository provides custom slash-command agents that Cursor can surface from the project root.

- `/content-marketer` → `.claude/agents/content-marketer/SKILL.md`
- `/customer-success-manager` → `.claude/agents/customer-success-manager/SKILL.md`
- `/context-manager` → `.claude/agents/context-manager/SKILL.md`

If Cursor does not immediately show these commands, reload the workspace or restart Cursor so it refreshes project agents.
