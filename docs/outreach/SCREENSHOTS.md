# Screenshots — nudge email mockups

## Full-page capture (Chrome)

1. Run `npm run dev` and open e.g. `http://localhost:3000/outreach-mockups/biweekly-digest.html`
2. Open **DevTools** (⌥⌘I / F12)
3. ⌘⇧P (Command Palette) → type **Capture full size screenshot** → Enter  
   Or: DevTools **⋮** menu → **Run command** → **Capture full size screenshot**

Repeat for `daily-adhoc.html`, or use **`nudge-emails-both-full.html`** to scroll one long page with both mockups embedded (iframe heights are approximate; adjust in `nudge-emails-both-full.html` if content clips).
